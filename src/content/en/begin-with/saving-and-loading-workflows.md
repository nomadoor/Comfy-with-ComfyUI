## Saving Workflow (In Browser)

Save the created workflow inside ComfyUI.

- 1. Click `Workflow` in the header and select `Save` or `Save as`
- 2. Give it a name you like and `Confirm` (or `Enter` key)

## Exporting Workflow (Save as File)

Export the workflow as a `.json` file for sharing or backup.

### Export as JSON File

- 1. Click `Workflow` in the header and select `Export`
- 2. Give it a name you like and `Confirm` (or `Enter` key)
- 3. Select the save destination and save

![](https://gyazo.com/e13e29d51a26ea2a29ec87cc872bf522){gyazo=loop}

### Embed in Generated Image and Save

Images generated by ComfyUI's standard image saving nodes (such as `Save Image`) automatically have workflow metadata (setting information) embedded. Just by loading this image, you can reproduce the workflow at the time of generation.


### Save as Screenshot

If you install custom nodes such as **[ComfyUI-Custom-Scripts](https://github.com/pythongosssss/ComfyUI-Custom-Scripts)**, you can take a screenshot of the entire workflow and save it with metadata embedded.

![](https://gyazo.com/a66d20bf36c02fa63c6cd5ab957fe4db){gyazo=loop}

## Loading Workflow

### Loading Saved Workflow

Open a workflow saved inside ComfyUI.

- 1. Click the folder icon ðŸ“‚ (or `W` key) on the left sidebar
- 2. Click the workflow you want to open from the list

![](https://gyazo.com/7c2149e7af5d6f78a30c9c03ff671356){gyazo=loop}

### Loading External Files (JSON / Image)

Load `.json` files or images with metadata from your PC.

**Method A: From Menu**
- 1. Click `Workflow` in the header and select `Open` (or `Ctrl + O`)
- 2. Select the file and open

**Method B: Drag & Drop**
- Drag & Drop the `.json` file or metadata-embedded image onto the ComfyUI canvas

![](https://gyazo.com/bbd3e9f3833a08a9af16bc3625c4747a){gyazo=loop}


### Load from Text

If you have copied text data in JSON format, you can load it just by pasting (`Ctrl + V`) on ComfyUI.
